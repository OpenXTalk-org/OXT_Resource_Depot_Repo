CalendarPopUp v1.5 is a pop-up Calendar Date Selector custom control by Tim BoboIn 2012 this was an update of the an OLD stack by Bob Cole, which shows how the Unix date functions can be called. In 2021 I replace the unix command with a few lines of script, which is how I should have done it from the begning. So this is no longer based on Bob's example. You can use and change this update freely. The calendar will look best if you have the font Helvetica Neue Condensed Medium installed.
